# Sales Data (Fake)

Dữ liệu giả lập về hoạt động kinh doanh, nhân khẩu học khách hàng và hiệu quả marketing, được tạo bởi AI Agent.

## Các file dữ liệu chính
- `sales_data.csv`: Chi tiết từng đơn hàng (Mã đơn, Khách hàng, Sản phẩm, Ngày đặt, Doanh thu, Vùng miền, Kênh bán, Chiến dịch).
- `customer_demographics.csv`: Thông tin khách hàng (Tuổi, Giới tính, Nghề nghiệp, Thu nhập, Vùng miền, Điểm Loyalty).
- `product_catalog.csv`: Danh mục sản phẩm (Mã hiệu, Giá vốn, Giá bán, Tồn kho, Ratings).
- `campaign_performance.csv`: Kết quả các chiến dịch marketing (Ngân sách, Lượt Click, Chuyển đổi, Doanh thu, ROI).
- `regional_sales.csv`: Dữ liệu kinh doanh và chỉ số vĩ mô theo vùng miền và thời gian.
- `sales_channels.csv`: Hiệu quả doanh kinh doanh phân phối theo từng kênh (Online, Offline).
