# InfiAgent Data

Dữ liệu được lấy từ [InfiAgent/InfiAgent](https://github.com/InfiAgent/InfiAgent/tree/main/examples/DA-Agent/data).

## Cấu trúc thư mục
- `da-dev-questions.jsonl`: Danh sách các câu hỏi mẫu để test agent.
- `da-dev-labels.jsonl`: Đáp án (ground truth) cho các câu hỏi.
- `da-dev-tables/`: Thư mục chứa 68 file CSV đa dạng chủ đề dùng làm database cho agent truy vấn.
